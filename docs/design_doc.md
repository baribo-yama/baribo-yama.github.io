# サイト構成
- ポートフォリオサイト
- 過去の制作物とつなげる
- ブログ機能
- ヘッダー、フッター
- フッターには自分へのコンタクト（連絡先とか）
- サイトはgithub.pagesで公開するため静的である必要がある

## ヘッダー
ヘッダーから以下の二つにアクセスできるようにする
- ホーム
- ブログ

## ホーム画面
ホーム画面は以下の要素で構成する
- 画面上部
  - ウェルカムボード（インパクトでひきつける
  - 簡単な自己紹介
- 画面中部
  - スキル紹介
  - 過去の制作物紹介
- 画面下部
  - コンタクト

## ディレクトリ構成
src/
├── components/
│   ├── layout/

│   └── blog/
│       ├── PostList.tsx      # 記事一覧

│   ├── Home.tsx              # ホーム画面（各 home/ コンポーネントを並べる）
